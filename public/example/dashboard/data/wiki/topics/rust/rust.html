Rust

Synthesis of the Rust references we consult. The defining feature is the ownership model — see rust-ownership — which gives memory safety without a garbage collector. Each value has a single owner; borrows are checked at compile time so use-after-free and data races become type errors.

  • Prefer Result<T, E> over panics for recoverable errors.
  • Reach for iterators and ? before hand-rolled loops and match ladders.
